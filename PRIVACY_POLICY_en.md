# Privacy Policy – ResumeMyLinkedin

Last updated: January 19, 2026

**ResumeMyLinkedin** is a browser extension that allows users to generate a DOCX resume based on information from **their own LinkedIn profile**.

This Privacy Policy is written in accordance with the **Chrome Web Store Developer Program Policies**.

---

## 1. Data collection and usage

ResumeMyLinkedin accesses **only personal data that is explicitly visible on the user’s own LinkedIn profile**, including but not limited to:

- Name  
- Headline (professional summary)  
- Location  
- Professional experience  
- Education  
- Volunteering activities  
- Contact information (such as email, phone number, and LinkedIn profile URL), **only when the user explicitly opens the “Contact info” section**

This data is used **exclusively to generate a resume (DOCX file)** requested by the user.

---

## 2. Local data processing

All information accessed by ResumeMyLinkedin is:

- Processed **locally in the user’s browser**
- Used only during the resume generation process
- Discarded immediately after the operation is completed

ResumeMyLinkedin **does not send, store, or process any personal data on external servers**.

---

## 3. Data sharing

ResumeMyLinkedin **does not share personal data with third parties**, including but not limited to:

- Companies
- Analytics platforms
- Advertising services
- External APIs

No personal data is sold, transferred, or reused for any other purpose.

---

## 4. Data that is NOT collected

ResumeMyLinkedin **does not collect**:

- Data from third-party LinkedIn profiles
- Private messages
- Restricted or non-public LinkedIn content
- Sensitive personal information beyond what is visible on the user’s profile
- Browsing data outside LinkedIn

---

## 5. Browser permissions

ResumeMyLinkedin requests only the permissions strictly necessary for its functionality:

- **activeTab**  
  To access the content of the active LinkedIn tab when the user interacts with the extension.

- **scripting**  
  To extract information from the LinkedIn page DOM.

- **downloads**  
  To allow the generated DOCX resume to be saved locally on the user’s device.

These permissions are **never used in the background** and are triggered only by explicit user actions.

---

## 6. User control and consent

Users have full control over the extension:

- No data is accessed automatically
- Data extraction only occurs after explicit user interaction
- The resume is generated only upon user request

By using ResumeMyLinkedin, the user consents to the use of data as described in this Privacy Policy.

---

## 7. Open source and transparency

ResumeMyLinkedin is an **open-source project**.  
The complete source code is publicly available on GitHub, allowing anyone to review, audit, or contribute to the project.

---

## 8. Changes to this Privacy Policy

This Privacy Policy may be updated from time to time to reflect functional changes or regulatory requirements.  
The “Last updated” date at the top of this document will always indicate the most recent revision.

---

## 9. Contact

If you have any questions or concerns regarding privacy or the extension’s behavior, you may open an issue in the project’s official GitHub repository.

---

### Summary statement (for Chrome Web Store)

> ResumeMyLinkedin does not store, track, or share personal data.  
> All data is processed locally in the user’s browser solely for the purpose of generating a resume from the user’s own LinkedIn profile.
